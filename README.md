# Calculadora Método de Gauss-Seidel

Aplicación desarrollada con Streamlit que permite resolver sistemas de ecuaciones lineales 3x3 mediante el método de Gauss-Seidel.

## Características
- Ingreso de coeficientes dinámico
- Verificación de diagonal dominante
- Iteraciones automáticas
- Historial de resultados
- Comprobación final

## Tecnologías
- Python
- Streamlit
- NumPy
- Pandas

## Cómo ejecutar

```bash
git clone https://github.com/SofiaCastillo13/CalculadoraGaussSeidel
pip install -r requirements.txt
streamlit run gaussseidel.py
